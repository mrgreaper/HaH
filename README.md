==========================

A terrible game for terrible Altspacers.


Setup
-----

1. Install Node.js, if not already installed.

2. Install dependencies
	```
	$ npm install
	```

3. Configure server by copying `config.sample.json` to `config.json`.

4. Build client JS
	```
	$ npm run build
	```

5. Start server
	```
	$ node server/app.js
	```
